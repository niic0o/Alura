Proyecto Café Aroma: Descubriendo el stack MERN

¿Qué es el stack MERN?

El stack MERN (MongoDB, Express.js, React y Node.js) es la base de este proyecto. Sin embargo, no se trata solo de tecnología, sino de una arquitectura bien pensada para escalabilidad, seguridad y rendimiento.

Microservicios y Modelo de Capas

Hemos elegido una arquitectura híbrida, aprovechando las ventajas de los microservicios para escalabilidad e integración con Mercado Pago. Para la seguridad, utilizamos JWT (JSON Web Token) y bcrypt para el hash de contraseñas con 10 saltos.

Gestión de Datos con MongoDB

Para la base de datos, utilizamos MongoDB, un motor NoSQL que nos permite gestionar datos de manera eficiente y escalable.
